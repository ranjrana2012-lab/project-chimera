# Project Chimera Documentation Index

This index reflects the current public Phase 1 boundary. The maintained public
route is the local Student / Laptop demonstrator, not an operational
multi-service production deployment.

Start here:

- `README.md` for the public project overview.
- `docs/README.md` for the documentation hub.
- `docs/guides/STUDENT_LAPTOP_SETUP.md` for the default setup route.
- `docs/DEVELOPER_SETUP.md` for contributor orientation.
- `docs/demo/README.md` for the current demo flow.

Advanced DGX Spark, Kimi, and service-compose material is available only when
the local evidence and credentials match those routes. Private grant evidence,
recordings, receipts, invoices, contact data, and local logs stay outside
public git.
