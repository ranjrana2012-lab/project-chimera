# Getting Started

Use the current public Phase 1 setup path:

- `docs/guides/STUDENT_LAPTOP_SETUP.md`

That guide runs the local operator-console demonstrator first and does not
require private API keys, grant material, or generated evidence files.

For the web demo, follow:

- `docs/demo/README.md`
- `docs/demo/troubleshooting.md`

For contributor setup, follow:

- `CONTRIBUTING.md`
- `docs/DEVELOPER_SETUP.md`

Do not commit `.env` files, receipts, invoices, screenshots, recordings, logs,
participant data, or grant records. Run `scripts/privacy_preflight.py` before
publishing public changes.
