# Architecture Overview

Project Chimera Phase 1 is a local adaptive AI demonstrator:

1. Audience-style text is entered through the CLI or web route.
2. The operator-console core classifies sentiment.
3. The strategy selector chooses an adaptive response mode.
4. The response, caption-style output, and export state are exposed locally.

The default validation path is `docs/guides/STUDENT_LAPTOP_SETUP.md`.
Advanced service-compose, DGX Spark, and Kimi routes require separate local
evidence before being presented as current behaviour.
