# Architecture

The current public architecture docs are intentionally narrow and describe the
Phase 1 local adaptive AI demonstrator.

Start with:

- `docs/architecture/overview.md`
- `docs/architecture/services.md`
- `docs/guides/STUDENT_LAPTOP_SETUP.md`

Older architecture decision records, diagrams, and service-topology documents
were removed from the public source set because they described unvalidated or
historical service plans as if they were current.
