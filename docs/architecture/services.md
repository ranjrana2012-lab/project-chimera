# Services

Current public Phase 1 service surface:

- `services/operator-console/chimera_core.py` - CLI demonstrator and adaptive
  response logic.
- `services/operator-console/chimera_web.py` - local web dashboard and demo API.
- `scripts/privacy_preflight.py` - public/private safety gate before publishing.
- `scripts/capture_phase1_evidence.py` - local evidence capture helper.

Other service directories are experimental or advanced routes unless their
specific guide says otherwise. Do not describe them as public production
behaviour without fresh validation evidence.
