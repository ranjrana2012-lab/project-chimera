# MVP Overview

This page is retained as a compatibility pointer. The previous MVP overview
mixed dated service inventory, deployment assumptions, and validation counts
that no longer represent the current public Phase 1 claim.

Current public scope:

- local adaptive AI demo through the operator-console CLI and web route;
- sentiment-driven strategy selection;
- caption-style output and export checks;
- private evidence capture kept out of public git.

Use these maintained references:

- `README.md` for the current project status.
- `docs/guides/STUDENT_LAPTOP_SETUP.md` for the default local route.
- `docs/demo/README.md` for the current demo script.
- `docs/reports/README.md` for maintainer validation history.

Advanced service and hardware routes should be validated separately before
being presented as current public behaviour.
