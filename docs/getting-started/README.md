# Getting Started

The old tracked student onboarding package has been removed from the public
source set because it mixed archived local onboarding material, dated demo
scripts, and obsolete infrastructure assumptions.

Use the current public setup paths instead:

- `docs/guides/STUDENT_LAPTOP_SETUP.md` for the default Student / Laptop route.
- `docs/DEVELOPER_SETUP.md` for contributor setup and repository orientation.
- `docs/demo/README.md` for the current Phase 1 demo flow.
- `docs/demo/troubleshooting.md` for Phase 1 demo troubleshooting.

Private course notes, live schedules, contact lists, and grant evidence should
stay in private storage, not in public git.
