# Student Tutorials

Welcome to the Project Chimera video tutorial series! These tutorials provide step-by-step guidance for common development tasks.

---

## Available Tutorials

### 1. Environment Setup Tutorial
**Duration:** 15-20 minutes
**Target:** New students joining Project Chimera

Covers:
- Prerequisites check (Python, Docker, kubectl, Git)
- Cloning the repository
- Running automated bootstrap (k3s installation)
- Verifying deployment
- Accessing services (Operator Console, Grafana)
- Running tests
- Setting up development tools
- First contribution workflow

**[Read Tutorial →](01-environment-setup.md)**

---

### 2. Testing Guide Tutorial
**Duration:** 20-25 minutes
**Target:** Students learning to write tests

Covers:
- Testing overview (unit, integration, e2e)
- Writing unit tests with pytest
- Running tests and checking coverage
- Using mocking and fixtures
- Writing integration tests
- Troubleshooting failed tests
- Testing best practices

**[Read Tutorial →](02-testing-guide.md)**

---

### 3. Debugging Guide Tutorial
**Duration:** 15-20 minutes
**Target:** Students troubleshooting issues

Covers:
- Common issues and solutions
- Service not starting
- High memory/CPU usage
- Service not responding
- Local debugging with pdb and VS Code
- Kubernetes debugging (pod exec, logs, events)
- Network debugging
- Performance profiling
- Debugging checklist
- Getting help

**[Read Tutorial →](03-debugging-guide.md)**

---

## Tutorial Format

Each tutorial includes:
- **Video script** - Can be used as screencast script or self-study guide
- **Code examples** - Copy-paste ready code snippets
- **Visual aids** - Diagrams and screenshots described
- **Timeline markers** - Time estimates for each section
- **Prerequisites** - What you need before starting
- **Resources** - Links to related documentation

---

## Using These Tutorials

### For Self-Study

1. Follow tutorials in order: Environment → Testing → Debugging
2. Follow along with the commands in your terminal
3. Pause and re-read sections as needed
4. Try the exercises and examples

### For Video Creation

Each tutorial is formatted as a complete video script:

```markdown
### Part 1: Introduction (0:00-1:00)

**Visual:** Title card with logo

**Speaker:** "Welcome to the tutorial..."

**Visual:** Screen recording showing...
```

Use with:
- Screen recording software (OBS, Camtasia)
- Text-to-speech for voiceover
- Or record your own voice
