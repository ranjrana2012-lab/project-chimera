# Contributing Documentation

The maintained contributor guide lives at the repository root:

- `CONTRIBUTING.md`

Use these public setup references with it:

- `docs/guides/STUDENT_LAPTOP_SETUP.md` for the default local route.
- `docs/DEVELOPER_SETUP.md` for repository orientation.
- `docs/github/REPOSITORY_SETTINGS.md` for maintainer-owned GitHub settings.
- `docs/github/BRANCH_CLEANUP_RUNBOOK.md` for post-merge branch cleanup.

Archived classroom onboarding schedules, contact lists, and email templates do
not belong in public git.
