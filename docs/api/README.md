# API Documentation

The current public API surface for Phase 1 is the local operator-console web
route.

Use:

- `docs/api/operator-console.md`
- `docs/guides/STUDENT_LAPTOP_SETUP.md`
- `docs/demo/README.md`

Older multi-service API catalogs were removed from the public source set because
they mixed future service designs, dated ports, and unvalidated operational
claims.
