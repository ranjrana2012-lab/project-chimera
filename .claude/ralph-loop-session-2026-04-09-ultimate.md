```bash

./scripts/health-check.sh --service all
```

### Performance Testing

```bash
# Full benchmark suite
python tests/performance/benchmark_phase2.py --service all --clients 50 --requests 500

# Stress test
python tests/performance/benchmark_phase2.py --service dmx --stress-test
```

### Security Audit

```bash
# Comprehensive security audit
./scripts/security-harden.sh --action audit --service all

# Compliance check
./scripts/security-harden.sh --action compliance
```

```bash
# Build all services
```
