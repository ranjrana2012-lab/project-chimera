## Editing Procedure

### Step 1: Rough Cut (30-45 minutes)

**Import**:
1. Create new project
2. Import all 7 scene files
3. Set project settings (1920x1080, 30fps)
4. Create timeline with 3-minute target

**Arrange Scenes**:
1. Drag scenes to timeline in order
2. Trim to exact timings from demo_script.md
3. Add 1-second transitions between scenes
4. Ensure smooth flow

**Trim to Target**:
- Scene 1 (Intro): 0:20
- Scene 2 (Positive): 0:30
- Scene 3 (Negative): 0:30
- Scene 4 (Neutral): 0:30
- Scene 5 (Compare): 0:30
- Scene 6 (Caption): 0:20
- Scene 7 (Outro): 0:20
- **Total**: 3:00

---

### Step 2: Audio Enhancement (20-30 minutes)

**Audio Normalization**:
1. Select all audio tracks
2. Normalize to -16 LUFS
3. Apply compression if needed
4. Ensure consistent levels

**Noise Reduction**:
1. Apply noise reduction filter
2. Remove background hum
3. Clean up any clicks/pops
4. Test on different speakers

**Voiceover Recording** (if live audio poor):
1. Record fresh narration in quiet room
2. Use good microphone (USB mic minimum)
3. Speak clearly and at moderate pace
4. Sync with video timeline

---

### Step 3: Visual Enhancement (30-45 minutes)

**Title Overlay**:
```
PROJECT CHIMERA
AI-Powered Adaptive Live Theatre Framework
Demonstration: April 9, 2026
```

**Text Highlights**:

Add text overlays for key points:
- Scene 2: "POSITIVE sentiment detected"
- Scene 3: "NEGATIVE sentiment detected"
- Scene 4: "NEUTRAL sentiment detected"
- Scene 5: "ADAPTIVE vs NON-ADAPTIVE"
- Scene 6: "ACCESSIBILITY: Caption formatting"

**Highlight Style**:
- Font: Sans-serif (Arial, Helvetica)
- Size: 48-60pt
- Color: Yellow with black outline
- Position: Top or bottom third
- Duration: 3-5 seconds each

**Color Correction** (if needed):
- Adjust brightness/contrast
- Ensure text is readable
- Consistent color temperature
- Professional appearance

---

### Step 4: Transitions (10-15 minutes)

**Transition Types**:
1. **Fade** (between scenes): 0.5 seconds
2. **Cut** (within scenes): Immediate
3. **Dip to Black** (section breaks): 1 second

**Transition Placement**:
- Scene 1→2: Fade
- Scene 2→3: Fade
- Scene 3→4: Fade
- Scene 4→5: Fade
- Scene 5→6: Fade
- Scene 6→7: Fade
- Scene 7→End: Fade to black

---

### Step 5: Background Music (Optional, 15-20 minutes)

**Music Selection**:
- Style: Gentle, ambient, electronic
- Mood: Professional, innovative
- Duration: 3:00 (looping if needed)
- Volume: -20dB (under narration)

**Music Placement**:
- Intro: Music starts, fades under narration
- Scenes: Music continues at low volume
- Outro: Music swells, fades at end

**Free Music Sources**:
- YouTube Audio Library (free use)
- Bensound (free with attribution)
- Incompetech (free with attribution)
- Audionautix (free)

---

### Step 6: Accessibility (10-15 minutes)

**Captions** (for accessibility):
1. Export narration transcript
2. Create caption file
3. Add captions to video
4. Verify timing and readability

**Caption Style**:
- Font: Clear, large (48pt+)
- Background: Semi-transparent black
- Text: White or yellow
- Duration: Sync with audio
- Position: Bottom center

## Troubleshooting

### Common Issues

**Issue**: Audio out of sync
**Solution**: Re-align audio track, adjust offset

**Issue**: Text not readable
**Solution**: Increase font size, add outline/background

**Issue**: File too large
**Solution**: Lower bitrate, reduce resolution to 1280x720

**Issue**: Export taking too long
**Solution**: Lower quality settings, be patient

**Issue**: Video crashes player
**Solution**: Re-export with different codec/settings
