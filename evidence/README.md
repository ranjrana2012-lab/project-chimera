# Public Evidence Placeholders

This directory is for public-safe placeholder structure only.

Do not commit generated screenshots, recordings, logs, receipts, invoices,
participant data, private grant records, tokens, or `.env` files here. Store real
evidence privately and run `scripts/privacy_preflight.py` before publishing.

For the private evidence checklist, see
`../docs/closeout/EVIDENCE_PACK_INDEX.md`.
